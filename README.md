# dotfiles

## requirements

- neovim
- ripgrep
- fzf
- eza
- uv
- nodejs
- npm
- cargo

`~/.config/chezmoi/chezmoi.toml`

```
[data]
	email = "my_personal_email@gmail.com"
	work = false
```

### Installed packages (cargo)

```
cargo install ripgrep tealdeer fd-find eza
```

pacman -Qe | wl-copy
